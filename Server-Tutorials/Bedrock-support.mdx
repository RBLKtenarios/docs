## ðŸš€ Step-by-Step Guide

### 1. Download Geyser and Floodgate

Download the latest versions of:

- `Geyser-Spigot.jar`
- `Floodgate-Spigot.jar` _(optional)_

ðŸ”— Official GeyserMC download:\
https://geysermc.org/download

---

### 2. Install Geyser on Your Server

1. Stop your server.
2. Upload `Geyser-Spigot.jar` to the `/plugins/` folder.
3. Start the server to generate config files.
4. A folder named `/plugins/Geyser-Spigot/` will be created.

---

### 3. Configure Geyser Settings

Edit `plugins/Geyser-Spigot/config.yml`:

```yaml
bedrock:
  address: 0.0.0.0
  port: 19132

remote:
  address: 127.0.0.1
  port: 25565
  auth-type: floodgate
```
